Understanding HTML Special Characters

Learn how to use HTML entities to display special characters correctly.

Ampersand (&) Less Than (<) / Greater Than (>) Copyright (©) Registered Trademark (®) Other Special Characters

The Ampersand (&)

To display an ampersand in HTML, use &amp;. Example: &amp; renders as &.

Less Than (<) and Greater Than (>)

To display a less than sign, use &lt;. Example: &lt; renders as <.

To display a greater than sign, use &gt;. Example: &gt; renders as >.

Copyright Symbol (©)

To display the copyright symbol, use &copy;. Example: &copy; 2024 renders as © 2024.

Registered Trademark Symbol (®)

To display the registered trademark symbol, use &reg;. Example: &reg; renders as ®.

Other Special Characters

  • Non-breaking space: &nbsp; renders as   (a space that will not break into a new line).
  • Quotation mark: &quot; renders as ".
  • Single quote: &apos; renders as '.
  • Euro sign: &euro; renders as €.
  • Trademark symbol: &trade; renders as ™.
